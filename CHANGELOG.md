# Changelog

All notable changes to Virtual Fishtank.

## [1.1.0] - 2025-02-10

### Added
- CHANGELOG.md to track project changes
- Verified all interactive features work

### Features Working
- Fish swimming with realistic movement physics
- Click fish to scare them (dart away)
- Add fish button (up to 30 max)
- Feed fish - all swim to food particle
- Fullscreen mode (press F)
- Keyboard shortcuts: F (fullscreen), Space (feed), + (add fish)
- Bubbles rising from bottom
- Seaweed swaying animation
- Rocks and decorations (coral, treasure chest)
- Water surface shimmer effect
- Light rays animation

### Technical Notes
- Pure HTML/CSS/JavaScript - no dependencies
- Runs as screensaver-style app
- Mobile responsive

## [1.0.0] - 2025-02-03

### Initial Release
- Animated fish aquarium screensaver
- Multiple fish species (emojis)
- Interactive controls
- Fullscreen support
