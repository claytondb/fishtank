# 🐠 Virtual Fishtank

> A calming virtual fishtank screensaver — now installable as a PWA!

**[▶️ View Demo](https://claytondb.github.io/fishtank/)**

## ✨ Features

- **Relaxing Aquarium** - Watch colorful fish swim around your screen
- **Interactive** - Click fish to make them dart away
- **Feed the Fish** - Drop food and watch them swarm
- **Fullscreen Mode** - Perfect for screensaver use
- **PWA Support** - Install on your device for offline use
- **No Dependencies** - Pure HTML/CSS/JavaScript

## 🎮 Controls

| Key/Action | Effect |
|------------|--------|
| **F** | Toggle fullscreen |
| **Space** | Feed fish |
| **+** | Add more fish |
| **Click fish** | Fish darts away |

## 🌊 Atmosphere

- Animated water surface with light rays
- Rising bubbles
- Swaying seaweed
- Coral and treasure chest decorations
- Sandy bottom with rocks

## 🐟 Fish Behavior

- Natural swimming with random direction changes
- Flocking toward food
- Boundary awareness
- Max 30 fish (starts with 8)

## 📱 Install as App

This is a Progressive Web App (PWA)! On supported browsers:
1. Click the install button in your browser's address bar
2. Or use "Add to Home Screen" on mobile
3. Enjoy as a standalone app, even offline

## 🚀 Usage

Open `index.html` in any modern browser.

For screensaver use, press **F** for fullscreen and let it run!

## ⚡ Tech Stack

- Pure HTML/CSS/JavaScript
- CSS animations for water effects
- requestAnimationFrame for smooth fish movement
- PWA with Web App Manifest
- No external dependencies
